## 1.6 目录结构

```text

phpxl 项目外层目录
│  
├─ project 项目总包 
│     ├─ default 默认的项目
│             ├─cli     脚本目录
│             ├─config  配置文件目录
│             ├─dataset 数据源目录
│             ├─event   事件目录
│             ├─func    自定义扩展函数目录
│             ├─iapi    与插件通信接口目录
│             ├─logic   逻辑层目录
│             ├─model   数据模型目录
│             ├─module  路由层目录
│             ├─plugin  插件目录
│             ├─point   埋点目录
│             ├─rpc     rpc目录
│             ├─task    任务节点目录
│             ├─www     网站根目录
│                 ├─static  js及图片目录
│                 ├─third   第三方sdk包
│                 ├─view    模版目录
│                 ├─clipure.php cli模式下入口文件
│                 ├─index.php   网站统一入口文件
│
├─xlcloud xl框架目录
│
```
